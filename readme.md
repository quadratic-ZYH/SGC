# SGC: Soft Gradient Collaboration for Backdoor Attacks in Self-Supervised Distillation

This repository provides the implementation of **SGC**, a soft gradient collaboration framework for backdoor attacks in self-supervised knowledge distillation.

## Environment

- Python 3.8
- PyTorch 1.13.1
- CUDA 11.7

Install dependencies:

```bash
pip install -r requirements.txt
```
